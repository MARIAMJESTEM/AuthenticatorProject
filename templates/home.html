Authenticator page

The goal of this project is to develop a system that performs person authentication based on facial recognition. The system uses a siamese neural network, a type of artificial intelligence, to compare a person's face from a database to a newly taken photo in real-time. The siamese neural network calculates a similarity score between the two images, which is used to authenticate the person's identity. This technology can be applied to various industries, such as security systems and access control, to improve efficiency and security.
